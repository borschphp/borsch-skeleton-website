Borsch Skeleton Help

Installation

Requirements

  • PHP 8.2+

  • Composer

Create a new project

Use the Composer's create-project command:

composer create-project borschphp/skeleton <app-name>

Run it

composer serve

Then go to 0.0.0.0:8080 to see the welcome page of your project.

It is also possible to run it via FrankenPHP, in worker mode.
You'll need to have Docker installed and up to use this command :

composer franken

Then go to https://localhost (where you'll need to accept self-signed certificate).
